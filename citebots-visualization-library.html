8. Card Container System

Nested card layouts with parent-child containers

Parent Card (Single Layer)

Clean white/gray background with icon header

This is a standard single-layer card with header icon, title, and content area. Used for simple widgets that don't require nested containers.

Parent Card (Two Layers)

With nested child container

Child Container

This gray-50/50 (light) or gray-700/30 (dark) container sits inside the parent card. It provides visual hierarchy and groups related content together. Commonly used for tables, charts, or grouped data.

Design Specifications

  • Parent Card Background: white (light) / gray-800 (dark)
  • Parent Border: gray-200/50 (light) / gray-700/50 (dark), 1px solid
  • Parent Corner Radius: rounded-2xl (16px)
  • Parent Padding: 1.5rem (24px)
  • Parent Hover: Enhanced border, shadow-lg, subtle lift effect
  • Header Icon Container: 40px × 40px, color-50/10 background, rounded-xl
  • Header Icon Size: 20px (w-5 h-5)
  • Child Container Background: gray-50/50 (light) / gray-700/30 (dark)
  • Child Corner Radius: rounded-xl (12px)
  • Child Padding: 1rem (16px)
  • Spacing: 1.5rem gap between header and child container

Design Principles

Color System

  • Primary Brand: Orange-500 (#f97316)
  • Success/Positive: Green-500/600
  • Warning/Alert: Orange-500/600
  • Error/Negative: Red-500/600
  • Info/Secondary: Blue-500/600
  • Neutral: Gray-400/500/600
  • Special Actions: Purple/Indigo-500/600

Typography

  • Font Family: System UI stack (native fonts)
  • Display: 4xl (36px), Bold (700)
  • Headings: xl-2xl (20-24px), Bold/Semibold (600-700)
  • Body: sm-base (14-16px), Regular (400) / Medium (500)
  • Small: xs-sm (12-14px), Regular/Medium
  • Labels: xs (12px), Medium (500), Uppercase

Spacing & Layout

  • Card Padding: 1.5rem - 2rem (24-32px)
  • Section Gaps: 1rem - 1.5rem (16-24px)
  • Component Spacing: 0.75rem - 1rem (12-16px)
  • Inline Elements: 0.5rem - 0.75rem (8-12px)
  • Border Radius: rounded-xl/2xl (12-16px) for cards
  • Border Radius: rounded-lg (8px) for nested items

Interactive States

  • Hover: Border opacity +50%, shadow-lg, subtle lift
  • Active: Scale 96% (pressed effect)
  • Transitions: 200ms duration, ease-out timing
  • Animations: 700ms-1s for data visualizations
  • Loading States: Spinning border animation
  • Focus: Ring-2 with brand color

This visualization library is designed for the Citebots platform.

All components are built with Tailwind CSS and support both light and dark modes.
